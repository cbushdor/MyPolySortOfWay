*MyPolymorphism.txt*   Extended "%" polymorphism

Enter in edit mode. Hence, type ':MyPolymorphism' and
check for outputs.
Check directory autoload/foo for extension covered.
Check with other extension that does not exist 
in autoload/foo (extension).vim. 
The aim is to test on extensions with polymorphism.

Tests can be made on opening files s.a:
>
   test.xml test.a test.cpp_
   test.cpp test.b test.cll
   test.c   test.c++
<

If extension does not exist error occured.
